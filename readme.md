# 线性规划与凸优化 作业一 第13题

## 输入方式
在文件 `input.txt`中输入所需要处理的数据
- 第一行为目标函数中的系数c，注意0也需要输入
- 第二行开始为每一条约束中变量前的系数以及对应的b

### 样例
```
6 14 13 0 0
1 4 2 1 0 48
1 2 4 0 1 60
```
- 该输入所对应的目标函数为 Max 6x1 + 14x2 + 13x3 + 0x4 +0x5
- 该输入所对应的约束为 
    - x1 + 4x2 + 2x3 + x4 = 48
    - x1 + 2x2 + 4x3 + x5 = 60
    
## 运行方法
- 在`input.txt`中输入数据
- 运行`Main.main()`
